Welcome to Go Web Server Demo

About This Project

This is a simple demonstration of a web server built with Go (Golang). It showcases different HTTP methods, form handling, and static file serving using Go's standard library.

Available Features

  • Static file serving
  • Form handling with POST method
  • Multiple endpoint demonstration
  • Basic routing implementation

Try It Out

Go to Form Demo Hello Endpoint

Available Endpoints

GET / - This index page

GET /hello - Returns a hello message

GET /form - Displays the form page

POST /form - Handles form submission

This server is built using Go's net/http package. Check out the source code to learn more about web server implementation in Go!
